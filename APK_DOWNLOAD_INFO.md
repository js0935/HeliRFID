## 📥 如何下載 APK

### 方法 1: 直接下載（推薦）

點擊上述每個版本的「直接下載連結」即可下載 APK 檔案。

### 方法 2: 從 GitHub Repository

1. 訪問: https://github.com/js0935/HeliRFID
2. 點擊 "Code" → "Download ZIP"
3. 解壓縮後找到 `releases/` 目錄
4. 找到對應版本的 APK 檔案

### 方法 3: 使用 Git Clone

```bash
git clone https://github.com/js0935/HeliRFID.git
cd HeliRFID/releases
# 在此目錄中找到所有版本的 APK 檔案
```

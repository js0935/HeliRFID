所有 APK 檔案位於: /releases/ 目錄

### GitHub Release
由於 gh command 不可用，無法自動創建 GitHub Release
用戶需要手動在 GitHub 網站上基于 tags 創建 release

創建步驟:
1. 訪問 https://github.com/js0935/HeliRFID/releases
2. 點擊 "Draft a new release"
3. 選擇 tag v2.0.0 或 v3.0.0
4. 上傳對應的 APK 檔案
5. 複製 RELEASE_NOTES.md 中的內容作為 release 說明
